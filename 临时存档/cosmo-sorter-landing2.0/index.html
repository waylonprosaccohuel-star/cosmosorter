常见问题

你可能想知道的一切

CosmoSorter 目前支持以下格式：

  • 文字：TXT、Markdown（.md）、Word（.docx）、纯文本粘贴
  • 图片：PNG、JPG/JPEG、WEBP、GIF（静态帧）、PDF（图片页）
  • 音频：MP3、WAV、M4A、OGG，最长支持 60 分钟单文件

我们持续扩展格式支持，如有特殊需求可通过反馈渠道告知我们。

我们非常重视创作者的知识产权与隐私安全：

  • 所有上传内容均经过 AES-256 加密存储，传输过程使用 TLS 1.3。
  • 您的素材不会被用于训练任何 AI 模型。
  • 每个世界观项目默认私密，仅您本人可见，除非您主动开启协作共享。
  • 您可以随时申请完整数据导出或账号注销（数据将在 30 天内彻底删除）。

可以！CosmoSorter 支持多种导出方式，方便与主流写作工具配合使用：

  • Markdown：兼容 Obsidian、Typora、Logseq 等笔记软件
  • Notion 导入包：一键导入 Notion 数据库（Pro 版）
  • PDF：适合打印或分享给合作者
  • API / Webhook：开发者可通过 API 将设定档案推送到自定义系统（Pro 版）

未来计划支持 Scrivener、Final Draft 等专业写作软件的直接集成。

在我们的内部测试中，基于真实创作者素材的标签分类准确率达到 98%（文字素材）和 91%（图片草图）。

影响准确率的因素包括：

  • 素材的清晰度与完整性（模糊草图识别率较低）
  • 世界观的独特性（越接近通用设定，准确率越高）

所有 AI 标签均支持一键手动修正，修正数据会帮助系统持续学习您的创作风格，准确率随使用时间提升。
